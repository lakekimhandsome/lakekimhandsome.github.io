# 김호수 | Portfolio

고등학생 개발자 김호수의 개인 포트폴리오 웹사이트입니다.

## 소개

HTML, CSS, JavaScript를 활용해 제작한 웹 프로젝트와 기술 스택, 자기소개를 담은 포트폴리오 사이트입니다.

## 기능

* Hero 섹션 (이름, 소개, GitHub / Projects 버튼)
* Projects 섹션 (프로젝트 카드 + Live Demo / GitHub 링크)
* Skills 섹션 (기술 스택 카드)
* About 섹션
* Contact 섹션
* 반응형 웹 디자인
* 스크롤 애니메이션 및 hover 효과

## 사용 기술

* HTML
* CSS
* JavaScript

## 프로젝트 구조

```
portfolio/
├── index.html
├── style.css
├── script.js
├── favicon.svg
└── README.md
```

## 로컬 실행

별도의 빌드 과정 없이 브라우저에서 `index.html`을 열면 됩니다.

```bash
# macOS
open index.html
```

또는 Live Server 등 정적 파일 서버를 사용해도 됩니다.

## GitHub Pages 배포

1. GitHub에 `portfolio` 저장소를 생성합니다.
2. 이 폴더의 파일을 저장소에 push합니다.
3. 저장소 **Settings → Pages**에서 Source를 `Deploy from a branch`로 설정합니다.
4. Branch를 `main`, Folder를 `/ (root)`로 선택 후 Save합니다.
5. 배포가 완료되면 `https://<username>.github.io/portfolio/`에서 확인할 수 있습니다.
