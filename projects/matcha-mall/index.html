프로젝트 소개

말차 제품을 소개하고 장바구니 기능을 체험할 수 있는 쇼핑몰 형태의 예시 웹사이트입니다. HTML, CSS, JavaScript만으로 프론트엔드 UI와 인터랙션을 구현했습니다.

프로젝트 목적

쇼핑몰 레이아웃, 상품 카드, 장바구니 흐름 등 실제 서비스에서 자주 쓰이는 UI 패턴을 연습하는 것이 목적이었습니다.

사용 기술

  • HTML
  • CSS
  • JavaScript

주요 기능

  • 상품 목록 및 상세 정보 표시
  • 장바구니 담기 및 수량 관리
  • 반응형 레이아웃

구현 과정

  • 상품 데이터 구조를 정의하고 페이지 레이아웃을 제작했습니다.
  • 장바구니 상태를 JavaScript로 관리하도록 구현했습니다.
  • 모바일 화면에 맞게 UI를 조정했습니다.

문제 해결 과정

  • 장바구니 수량 변경 시 UI가 즉시 반영되도록 DOM 업데이트 로직을 정리했습니다.
  • 그리드 레이아웃이 작은 화면에서 깨지는 문제를 미디어 쿼리로 해결했습니다.

배운 점

  • 쇼핑몰 UI 구조와 사용자 흐름 설계
  • JavaScript로 상태를 관리하는 기본 방법
