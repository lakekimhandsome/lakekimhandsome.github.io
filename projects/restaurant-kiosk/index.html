식당 키오스크

예시

메뉴 선택과 주문 흐름을 체험할 수 있는 식당 키오스크 예시 웹 애플리케이션입니다.

lakekim.com/restaurant-kiosk
GitHub

프로젝트 소개

식당 키오스크처럼 메뉴를 선택하고 장바구니에 담아 주문하는 흐름을 구현한 예시 프로젝트입니다. 터치 친화적인 UI를 목표로 제작했습니다.

스크린샷 갤러리

프로젝트 목적

키오스크 UI의 메뉴 선택, 옵션 추가, 주문 확인 같은 실사용 흐름을 웹으로 구현하며 인터랙션 설계를 연습하는 것이 목적이었습니다.

사용 기술

  • HTML
  • CSS
  • JavaScript

주요 기능

  • 메뉴 카테고리 및 상품 선택
  • 장바구니 및 주문 확인
  • 터치하기 쉬운 큰 버튼 UI

구현 과정

  • 키오스크 화면 흐름을 설계하고 UI를 구성했습니다.
  • 메뉴 데이터와 장바구니 상태를 JavaScript로 관리했습니다.
  • 주문 확인 화면과 초기화 기능을 추가했습니다.

문제 해결 과정

  • 장바구니 합계 계산 로직을 정리해 오류를 줄였습니다.
  • 작은 화면에서 버튼 크기와 간격을 조정해 사용성을 개선했습니다.

배운 점

  • 키오스크형 UI의 사용자 흐름 설계
  • 상태 기반 UI 업데이트 방법
