CafeRadar

운영 중 실제 서비스

카페 근무 환경 후기와 구인·구직 정보를 공유하는 커뮤니티형 웹 서비스입니다.

caferadar.lakekim.com
GitHub

프로젝트 소개

CafeRadar는 카페에서 일해본 사람들이 근무 환경 후기를 남기고, 카페에서 일하려는 사람들이 참고할 수 있는 커뮤니티 서비스입니다. 회원 인증, 후기 작성, 구인글 등록, 검색 기능을 Supabase와 연동해 구현했습니다.

프로젝트 목적

카페 알바를 구할 때 실제 근무 환경 정보를 미리 확인할 수 있는 플랫폼을 만드는 것이 목표였습니다. 단순 정보 제공을 넘어 회원 기반의 후기 작성과 구인·구직 정보 공유가 가능한 실사용 서비스를 구현하고자 했습니다.

사용 기술

  • HTML
  • CSS
  • JavaScript
  • Supabase

주요 기능

  • 이메일·비밀번호 기반 회원가입 및 로그인
  • 카페 근무 후기 작성·조회·검색
  • 구인·구직 글 등록 및 검색
  • 마이페이지에서 내 후기 관리
  • 모바일 반응형 UI

구현 과정

  • 서비스 흐름과 화면 구조를 설계한 뒤 메인 UI를 제작했습니다.
  • Supabase Auth와 PostgreSQL 테이블, RLS 정책을 설정했습니다.
  • 모듈형 JavaScript로 인증, 후기, 구인 기능을 분리해 구현했습니다.
  • 검색, 필터, 모달 UI를 추가하며 사용성을 개선했습니다.
  • GitHub Pages에 배포해 실제 접속 가능한 서비스로 공개했습니다.

문제 해결 과정

  • RLS 정책 미적용으로 데이터 접근 오류가 발생해 SQL 마이그레이션으로 정책을 재정비했습니다.
  • 회원 탈퇴 시 연관 데이터 삭제를 RPC 함수로 처리해 안정성을 높였습니다.
  • GitHub Pages 환경에서 Supabase Auth redirect URL을 맞추며 배포 이슈를 해결했습니다.

배운 점

  • 인증, DB, RLS를 포함한 풀스택형 웹 서비스 구조
  • 기능별 JS 모듈 분리와 유지보수 방법
  • 실제 사용자 흐름을 고려한 UI·UX 설계
