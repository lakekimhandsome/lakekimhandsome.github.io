SpeakLog

운영 중 실제 서비스

온라인 영어회화 서비스 홈페이지와 상담 신청, 후기·공지 관리를 위한 관리자 시스템을 함께 구축한 프로젝트입니다.

speaklog.co.kr
GitHub

프로젝트 소개

SpeakLog는 필리핀 강사와 함께하는 1:1 온라인 영어회화 서비스를 소개하고, 상담 신청을 받을 수 있는 공개 웹사이트입니다. 서비스 운영에 필요한 후기와 공지사항을 관리할 수 있는 관리자 페이지도 함께 구현했습니다.

프로젝트 목적

영어회화 서비스의 신뢰감 있는 첫인상을 전달하고, 방문자가 쉽게 상담을 신청할 수 있도록 하는 것이 목표였습니다. 또한 운영자가 코드 수정 없이 콘텐츠를 관리할 수 있는 구조를 만들고, 실제 도메인으로 서비스를 배포해 보는 경험을 목표로 했습니다.

사용 기술

  • HTML
  • CSS
  • JavaScript
  • Supabase
  • Custom Domain

주요 기능

  • 서비스 소개, 강사·수업 안내, FAQ 등 공개 페이지 구성
  • 상담 신청 폼 및 Supabase DB 저장
  • 수강 후기 조회 및 관리자 후기 관리
  • 공지사항 작성·수정·삭제
  • 관리자 로그인 및 대시보드
  • 모바일 반응형 UI

구현 과정

  • 서비스 구조와 페이지 흐름을 먼저 설계한 뒤 공개 페이지 UI를 제작했습니다.
  • Supabase에 상담, 후기, 공지 테이블과 RLS 정책을 설정했습니다.
  • 공개 페이지에서 API를 통해 데이터를 불러오고, 폼 제출 기능을 연결했습니다.
  • 관리자 전용 페이지를 분리해 인증 후 CRUD 기능을 구현했습니다.
  • 커스텀 도메인 연결과 SEO 메타 태그를 적용해 실제 서비스 형태로 배포했습니다.

문제 해결 과정

  • Supabase RLS 설정 오류로 데이터 조회가 되지 않을 때, 정책과 권한(grant)을 점검하며 해결했습니다.
  • 관리자 기능과 공개 API 권한을 분리해 보안과 사용성을 동시에 맞췄습니다.
  • 모바일에서 레이아웃이 깨지는 구간을 미디어 쿼리로 수정하며 반응형을 개선했습니다.

배운 점

  • 정적 웹사이트만으로도 Supabase를 활용하면 실제 운영 가능한 서비스를 만들 수 있다는 점
  • 공개 페이지와 관리자 페이지를 분리 설계하는 방법
  • 도메인 연결, SEO, 배포까지 포함한 전체 웹 서비스 흐름
