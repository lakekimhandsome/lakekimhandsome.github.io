운동일지

개발 중 iOS 앱

메모 앱보다 빠르게 운동을 남기기 위한 iOS 운동일지 앱입니다. 한 화면에서 세션 생성, 운동 추가, 세트 기록, 코멘트 작성까지 끝낼 수 있도록 설계했으며, Apple Watch 동반 앱도 함께 만들었습니다.

GitHub

프로젝트 소개

운동일지는 운동 기록에 집중한 iOS 앱입니다. 복잡한 분석 기능 없이, 운동 세션을 빠르게 만들고 세트 수를 기록하는 데 필요한 기능만 담았습니다. Capacitor로 React 웹앱을 iOS 네이티브 앱으로 패키징했고, 운동 중 손목에서 쓸 수 있는 Apple Watch 동반 앱도 함께 개발했습니다.

스크린샷 갤러리

이미지를 클릭하면 크게 볼 수 있습니다.

프로젝트 목적

기존 메모 앱으로 운동을 기록할 때 느껴지는 번거로움을 줄이는 것이 목표였습니다. 로그인 없이 바로 기록을 시작하고, 세트 추가 시 휴식 타이머가 자동으로 시작되는 등 실제 운동 중 손이 바쁠 때도 빠르게 쓸 수 있는 UX를 만들고자 했습니다. 워치에서도 세트 추가와 타이머 확인이 가능하도록 보완했습니다.

사용 기술

  • React
  • TypeScript
  • Vite
  • Capacitor
  • iOS
  • watchOS
  • SwiftUI

주요 기능

  • 한 화면에서 세션·운동·세트·코멘트 기록
  • 세트 추가 시 자동 휴식 타이머
  • Apple Watch에서 세트 추가 및 휴식 타이머 확인
  • 카테고리별 운동 분류 (PUSH/PULL/LEGS 등)
  • 이전 세션 참고 및 스와이프 제스처
  • 운동일지 내보내기·가져오기·초기화
  • 로컬 저장 (로그인 불필요)

구현 과정

  • React + TypeScript로 운동 기록 UI와 상태 관리를 구현했습니다.
  • Local Storage 기반으로 데이터를 저장하도록 설계했습니다.
  • Capacitor를 연동해 iOS 앱으로 빌드하고 햅틱 피드백을 추가했습니다.
  • WatchConnectivity로 iPhone과 Apple Watch 앱을 연동했습니다.

배운 점

  • 모바일 퍼스트 UX 설계와 터치 제스처 처리
  • React Context로 복잡한 상태를 관리하는 방법
  • Capacitor를 활용한 웹 → iOS 앱 변환 경험
  • iOS와 watchOS 간 데이터 동기화 경험
