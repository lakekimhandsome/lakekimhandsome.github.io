Bokshil Café

예시

감성적인 분위기의 카페를 소개하는 예시 랜딩 페이지입니다.

페이지 방문 GitHub

프로젝트 소개

카페 브랜드의 분위기와 메뉴, 공간을 소개하는 원페이지 랜딩 사이트 예시입니다. 감성적인 UI 표현에 초점을 맞췄습니다.

프로젝트 목적

랜딩 페이지에서 중요한 비주얼, 타이포그래피, 섹션 흐름을 연습하고 브랜드 분위기를 웹으로 표현하는 방법을 익히는 것이 목적이었습니다.

사용 기술

  • HTML
  • CSS
  • JavaScript

주요 기능

  • 카페 소개 및 메뉴 섹션
  • 스크롤 기반 섹션 구성
  • 반응형 레이아웃

구현 과정

  • 카페 컨셉에 맞는 색감과 폰트를 선정했습니다.
  • 히어로, 메뉴, 갤러리 섹션을 순차적으로 구성했습니다.
  • 스크롤 애니메이션과 hover 효과를 추가했습니다.

문제 해결 과정

  • 이미지와 텍스트 대비가 낮아 가독성이 떨어지는 구간을 조정했습니다.
  • 모바일에서 섹션 간격이 좁아 보이는 문제를 수정했습니다.

배운 점

  • 브랜드 분위기를 UI로 표현하는 방법
  • 랜딩 페이지 섹션 설계와 시각적 흐름
