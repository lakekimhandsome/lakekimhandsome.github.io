프로젝트 소개

학원의 소개, 수업 정보, 연락처 등을 보여주는 정적 홈페이지 예시 프로젝트입니다. 정보 전달형 웹사이트의 기본 구조를 연습했습니다.

스크린샷 갤러리

프로젝트 목적

기관·학원 홈페이지에서 자주 사용하는 섹션 구성과 타이포그래피, 레이아웃을 익히는 것이 목적이었습니다.

사용 기술

  • HTML
  • CSS
  • JavaScript

주요 기능

  • 학원 소개 및 강의 안내 섹션
  • 문의/연락처 정보 제공
  • 반응형 레이아웃

구현 과정

  • 페이지 섹션 구조를 설계하고 HTML 마크업을 작성했습니다.
  • CSS로 시각적 계층과 가독성을 맞췄습니다.
  • 간단한 인터랙션을 JavaScript로 추가했습니다.

문제 해결 과정

  • 섹션 간 여백과 정렬이 어색한 부분을 Flexbox/Grid로 조정했습니다.
  • 모바일에서 네비게이션이 겹치는 문제를 수정했습니다.

배운 점

  • 정보 전달형 웹사이트의 섹션 구성 방법
  • 가독성 있는 타이포그래피와 레이아웃 설계
